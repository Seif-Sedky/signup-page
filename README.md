# Discrete++ Signup Page

A clean, modern signup page designed for a discrete mathematics platform, showcasing a strong eye for UI/UX design and frontend development.

## Key Strengths Demonstrated

- **Responsive & Accessible Layout**  
  Semantic HTML5 structure combined with intuitive form inputs and labels ensures ease of use and accessibility.

- **Custom Styling & Design**  
  Utilizes CSS variables, custom fonts, and subtle hover/focus effects for a polished and professional look.

- **User Experience Focus**  
  Clear input validation styles improve user feedback, enhancing form usability.

- **Attention to Detail**  
  Thoughtful use of flexbox layouts, spacing, and typography for balanced visual hierarchy.

- **Clean Code Organization**  
  Separation of concerns with HTML and CSS, making the code easy to maintain and extend.
